ЗАМЕТКИ PWA

Простое прогрессивное веб-приложение для текстовых заметок с автоматической вставкой геокоординат.
Все заметки хранятся локально в IndexedDB, приложение работает офлайн.


КЛЮЧИ И ВНЕШНИЕ СЕРВИСЫ

- API-ключи не требуются. Приложение использует только встроенный Geolocation API браузера.
- Для работы координат пользователь должен дать разрешение на доступ к местоположению.
- Внешние сервисы не используются. Данные хранятся только на устройстве.


ТРЕБОВАНИЯ ДЛЯ ЗАПУСКА

- Любой современный браузер (Chrome, Edge, Firefox, Safari).
- Локальный веб-сервер. PWA-функциональность (Service Worker) требует протокол http:// или https://.
  

БЫСТРЫЙ СТАРТ (НА КОМПЬЮТЕРЕ)

1. Перейдите в папку проекта notes-pwa.
2. Запустите локальный сервер одной командой:

   npx http-server -a 0.0.0.0 -p 8080 -c-1

3. В браузере откройте http://localhost:8080


ПАРАМЕТРЫ КОМАНДЫ HTTP-SERVER

   -a 0.0.0.0   Слушать все сетевые интерфейсы 
   -p 8080      Порт 
   -c-1         Отключить кэширование на сервере


ДОСТУП С МОБИЛЬНОГО ТЕЛЕФОНА (В ЛОКАЛЬНОЙ СЕТИ)

1. Компьютер и телефон должны быть подключены к одной Wi-Fi сети.
2. Узнайте IP-адрес компьютера (команда ipconfig в Windows или ifconfig в macOS/Linux).
   Пример: 192.168.1.5.
3. Запустите сервер с параметром -a 0.0.0.0 (см. выше).
4. На телефоне откройте браузер и перейдите по адресу http://<IP-компьютера>:8080,
   например http://192.168.1.5:8080.



ДОСТУП ЧЕРЕЗ ИНТЕРНЕТ (ЕСЛИ ТЕЛЕФОН НЕ В ЛОКАЛЬНОЙ СЕТИ)

Можно использовать временный публичный туннель. Самый простой способ — localtunnel:

1. Убедитесь, что локальный сервер запущен на порту 8080.
2. В новом окне терминала выполните:

   npx localtunnel --port 8080

3. Программа выдаст ссылку вида https://xxxx.loca.lt. Откройте её на телефоне.
4. При первом переходе нажмите «Click to Continue» (защита от ботов).



УСТАНОВКА КАК PWA

После открытия сайта в браузере Chrome или YandexBrowser:
   - На телефоне: в меню браузера выберите «На экран «Домой»» / «Установить приложение».

После установки заметки будут доступны даже без подключения к сети.


СТРУКТУРА ПРОЕКТА

notes-pwa/

   style.css          – стили
   app.js             – логика приложения (IndexedDB, геолокация)
